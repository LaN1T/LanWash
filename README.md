# LanWash — приложение для записи на автомойку
Учебный проект на Flutter/Dart. Приложение позволяет клиентам записываться на автомойку, а администратору управлять записями и каталогом услуг.
- Платформы: iOS, macOS, Flutter Web (Chrome)

## Роли и функциональность
### Клиент
- Клиент может зарегистрироваться
- Войти в аккаунт
- Записаться на мойку через визард (выбор даты, типа мойки, доп. услуг, данных авто)
- Просматривать свои записи
- Добавлять услуги и акции в избранное
- Записываться по акции напрямую из раздела избранного.
### Администратор
- Администратор видит все записи всех клиентов
- Может редактировать и удалять записи
- Управлять каталогом услуг
- Просматривать избранные записи.
## Типы мойки
- Базовая (800₽, 30 мин) — активная пена, ручная очистка, сушка. 
- Комплексная (1500₽, 60 мин) — базовая мойка плюс уборка салона и пылесос, пылесосная уборка включена автоматически. 
- Премиум (3000₽, 90 мин) — комплексная мойка плюс уход за пластиком и резиной, чернение шин и ароматизация включены автоматически.
## Акции
- Акция недели: комплекс + ароматизация — 1600₽. 
- Весенняя акция: мойка + воск — 1500₽. 
- Выходной пакет: комплексная мойка −20% — 1100₽ (только сб/вс). 
- Пакет для внедорожников — 2000₽.
## База данных
Локальная SQLite через sqflite (мобайл/macOS) и sqflite_common_ffi (десктоп). Таблицы: appointments, extra_favorites, logs, promos, service_favorites, services, sqlite_master, sqlite_sequence, users. Поддержка миграций и логирования. Пароли хранятся в виде SHA-256 хэша.
## Стек
Flutter 3.x, Dart, Provider, sqflite, sqflite_common_ffi, intl, http, crypto
