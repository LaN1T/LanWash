# LanWash — приложение для записи на автомойку

Учебный проект на Flutter/Dart. Приложение позволяет клиентам записываться на автомойку, администратору управлять записями и каталогом услуг, а мойщикам видеть назначенные им записи.

**Платформы:** iOS, macOS, Android, Flutter Web (Chrome)

---

### Клиент
- Регистрация и вход в аккаунт
- Запись на мойку через трёхшаговый визард (дата/время → тип мойки и доп. услуги → подтверждение)
- Запись по акции напрямую из раздела акций или избранного
- Просмотр своих записей с актуальной ценой (если администратор изменил состав услуг — отображается новая и зачёркнутая старая цена)
- Редактирование профиля: имя, телефон, марка и номер авто, пароль
- Добавление услуг и акций в избранное
- Просмотр уведомлений (заметок от администратора)

### Администратор
- Просмотр всех записей всех клиентов
- Создание, редактирование и удаление записей
- Назначение мойщиков на записи (до 3 мойщиков на одну запись)
- Управление каталогом услуг
- Просмотр и очистка журнала действий (логов) с фильтрацией по пользователю
- Просмотр избранных записей
- Отправка заметок клиентам

### Мойщик
- Просмотр назначенных записей (только чтение)
- Детальная информация по каждой записи: клиент, авто, тип мойки, доп. услуги, цена, заметки
- Просмотр заметок от администратора
- Pull-to-refresh для обновления списка записей

## Типы мойки

| Тип | Цена | Время | Включено автоматически                        |
|---|---|---|-----------------------------------------------|
| Базовая | 800 ₽ | 30 мин | —                                             |
| Комплексная | 1 500 ₽ | 1 ч | Пылесосная уборка                             |
| Премиум | 3 000 ₽ | 1 ч 30 мин | Пылесосная уборка, Чернение шин, Ароматизация |

Авто-включённые услуги не прибавляются к цене и не учитываются повторно во времени.

---

## Акции

| Название | Состав | Цена | Ограничение |
|---|---|---|---|
| Акция недели: комплекс + ароматизация | Комплексная мойка + Ароматизация | 1 600 ₽ | — |
| Весенняя акция: мойка + воск | Базовая мойка + Нанесение воска | 1 500 ₽ | — |
| Выходной пакет: комплексная мойка −20% | Комплексная мойка со скидкой | 1 200 ₽ | Только сб/вс |
| Пакет для внедорожников | Комплексная мойка + Чернение шин + Обработка арок | 2 000 ₽ | — |

**Расчёт цены при записи по акции:**
- База = акционная цена (или basePrice × (100 − скидка%) для процентных акций)
- К базе прибавляются только дополнительные услуги, выбранные сверх акции
- Услуги, входящие в акцию, не прибавляются к цене и отображаются как «Задано акцией»

**Расчёт времени:**
- Берётся базовое время типа мойки
- К нему прибавляется время только тех доп. услуг, которые не входят в тип мойки автоматически

## Визард записи (трёхшаговый)

**Шаг 1 — Дата и время:** выбор даты (с ограничением только на выходные для соответствующих акций) и временного слота (09:00–18:00).

**Шаг 2 — Услуги и данные авто:**
- Выбор типа мойки (RadioListTile)
- Выбор доп. услуг (CheckboxListTile), отсортированных от дешёвых к дорогим
- Услуги, входящие в тип мойки, отмечены «Включено» и недоступны для снятия
- Услуги, заданные акцией, отмечены «Задано акцией» и также заблокированы
- Поля клиента и авто предзаполняются из профиля
- Поддержка избранных доп. услуг

**Шаг 3 — Подтверждение:**
- Итоговая цена с перечёркнутой ценой без акции (если есть скидка) и суммой экономии
- Суммарное время с учётом всех услуг

## Журнал действий (логи)

Все ключевые события фиксируются в таблице `logs`:
- Вход / выход из системы, неудачные попытки входа
- Регистрация
- Создание, редактирование, удаление записей
- Добавление/удаление из избранного (услуги и доп. услуги)
- Обновление профиля

Администратор видит журнал в разделе «Логи» с возможностью фильтрации по пользователю и полной очистки.

---

## Сессия и аутентификация

- Пароли хранятся в виде SHA-256 хэша
- После входа данные пользователя сохраняются в `SharedPreferences` — сессия переживает перезагрузку страницы (актуально для Flutter Web)
- При выходе сессия очищается

---

## База данных

Локальная SQLite через `sqflite` (iOS/macOS) и `sqflite_common_ffi` (десктоп/веб).

| Таблица | Назначение |
|---|---|
| `users` | Аккаунты пользователей (SHA-256 хэш пароля) |
| `appointments` | Записи на мойку |
| `services` | Каталог услуг и акций |
| `promos` | Акции (связаны с services) |
| `service_favorites` | Избранные услуги по пользователям |
| `extra_favorites` | Избранные доп. услуги по пользователям |
| `logs` | Журнал действий |
| `washer_notes` | Заметки администратора для мойщиков и клиентов |
| `deleted_notifications` | Уведомления об удалении записей администратором |

Поддержка инкрементальных миграций (`onUpgrade` по версии БД).

---

## Бэкенд

Python/FastAPI REST API с SQLite через `aiosqlite`.

**Базовый URL:** `http://localhost:8000/api`

**Запуск:** `uvicorn main:app --reload` (порт 8000)

Роутеры: `/auth`, `/appointments`, `/services`, `/logs`, `/notes`
